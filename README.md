# Employee Management API

A RESTful Employee Management API built with Python and FastAPI.

## Features

- GET all employees
- GET employee by ID
- POST create employee
- PUT update employee
- DELETE employee

## Technologies Used

- Python
- FastAPI
- Pydantic
- Uvicorn

## How to Run

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
